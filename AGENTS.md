# AWS Learning Projects

The purpose of this repository is to learn foundational AWS services and concepts by building different practical projects. 

## Core Guidelines
- Focus on building working, practical AWS implementations.
- Prioritize simplicity and clear architecture over complex, production-level edge cases.
- Explain the fundamental purpose of the AWS resources being created or configured.
- Adhere to basic cloud best practices (e.g., least privilege IAM, avoiding hardcoded credentials) to establish good habits.
